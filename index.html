Projects

Static Amazon Clone

This project involved the creation of a static Amazon website clone using HTML and CSS. The primary objective was to replicate the visual structure and styling of Amazon's homepage, demonstrating proficiency in front-end development. Utilizing semantic HTML, I constructed the site's layout, including the header, navigation bar, product grids, and footer. CSS was employed to style the elements, mirroring Amazon's design aesthetics and implementing responsive design principles. While lacking backend functionality, this project served as a valuable exercise in mastering core web development skills and understanding website layout and design.

React Music Player

This project involved the development of a responsive music player web application using React.js. The primary objective was to create a user-friendly interface for playing and managing audio files. React's component-based architecture was utilized to build modular and reusable UI elements, such as the player controls, playlist, and track information display. State management was implemented to handle audio playback, track selection, and volume control. Key features included audio playback controls (play, pause, next, replay), a dynamic playlist, track progress visualization.

Online Code Editor

This is a visual concept of my React-based Online Code Editor with HTML/CSS/JS support and live preview. A React-based online code editor that supports HTML, CSS, and JavaScript with real-time live preview. Users can write and test code directly in the browser and generate shareable links for their work. The app features a clean dark theme, tabbed interface for different languages, and a responsive layout for seamless user experience. Designed to be lightweight yet functional, it's a great tool for quick prototyping and learning.
